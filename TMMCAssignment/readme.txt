TMMCAssignment
---------------

What it does
-------------
This is a small Windows console app that counts the vertical black lines in the given JPEG images.

Build
-----
From the folder containing `TMMCAssignment.csproj`:
  dotnet build -c Release

Run
----
   - Run `TMMCAssignment.bat`
   - Enter full path for image (e.g. C:\full\path\img_1.jpg)
   - The app prints the line count as a number.
   - Enter an empty line (or `exit`) to quit.
